# JPineda IT Ops Lab

A hands-on IT Operations lab focused on automation, identity management, endpoint systems, and modern AI-driven workflows.

---

## Overview
This repository documents my approach to modern IT Operations, focusing on automation, identity management, endpoint systems, and scalable infrastructure.

The goal is to design and implement systems that reduce manual effort, improve security, and enable better operational decision-making.

---

## Core Areas
- Identity & Access Management (Okta, Azure AD)
- Endpoint Management (Jamf, Intune)
- Automation (PowerShell, workflows, onboarding/offboarding)
- SaaS & Collaboration Tools (Microsoft 365, Google Workspace, Slack, Zoom)
- AI-assisted operations (knowledge systems, workflow augmentation)
- Security & Compliance (MFA, access controls, audit readiness)

---

## Philosophy
IT Operations is evolving from reactive support into proactive system design and automation.

Similar to the shift to cloud infrastructure, AI and APIs are becoming core layers in how systems integrate, operate, and scale.

The focus is on:
- Reducing operational friction
- Increasing automation coverage
- Improving visibility through metrics
- Maintaining security and compliance

The goal is to move IT from ticket-driven work to systems that are automated, measurable, and scalable.
